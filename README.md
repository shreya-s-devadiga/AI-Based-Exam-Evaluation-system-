# AI-Based Exam Evaluation System

## 📌 Description
An AI-powered system that evaluates descriptive answers and generates marks with feedback.

## 🚀 Features
- Admin Dashboard
- Teacher Panel (Create Questions)
- Student Exam System
- AI-based Answer Evaluation
- Result & Feedback Generation

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Python
- Database: MySQL

## ⚙️ How to Run

### Backend
cd backend
pip install -r requirements.txt
python main.py

### Frontend
cd frontend
npm install
npm run dev
